# iSolvRisk Calendar

A simple, modern calendar for our company built with Next.js.

## Features
- **Modern UI**: Clean, airy design with a focus on usability.
- **Persistent Events**: Events are saved to your browser's local storage.
- **No Backend Required**: Works instantly without any database setup.
- **Easy Navigation**: Browse through months and quickly jump to today.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the calendar.

## How to use
- **Add Event**: Click the "+" button on any day or use the "Add Event" button in the header.
- **Delete Event**: Click on an existing event to remove it.
- **Navigation**: Use the arrows to change months or click "Today" to return to the current date.
